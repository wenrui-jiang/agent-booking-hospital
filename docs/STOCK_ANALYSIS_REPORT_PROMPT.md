# 个股分析报告 Prompt

请将下面内容复制给大模型，并把最后一行的股票名称或代码补全。

```text
你是一名专业证券研究员。请基于公开资料，为指定股票撰写一份结构完整、逻辑清晰、偏研究报告风格的个股分析报告。

请优先使用最新公开资料，包括但不限于：公司年报、季报、临时公告、交易所披露、公司官网、券商研报、财经数据平台和公开行情数据。所有财务数据必须注明时间口径，例如“2025年年报”“2026年一季度”。如果缺少最新数据，请明确说明“公开资料有限”或“需进一步核实”，不要编造数据。

报告必须包含以下部分：

1. 核心结论
- 用简洁语言概括公司当前投资价值。
- 判断公司属于成长型、周期型、困境反转型、价值型或高风险投机型标的。
- 给出明确但审慎的综合判断，例如：建议关注、谨慎观望、高风险参与、不建议参与等。
- 明确说明本文不构成投资建议。

2. 行业分析
- 说明公司所处行业、产业链位置和主要下游需求。
- 分析行业增长驱动因素，如政策、技术升级、国产替代、需求周期、资本开支等。
- 分析行业竞争格局，包括主要竞争者、行业集中度、进入壁垒和盈利能力。
- 说明行业当前面临的主要压力和风险。

3. 个股分析
- 介绍公司主营业务、核心产品或服务、收入结构。
- 分析公司的竞争优势，例如技术、品牌、渠道、客户、成本、资源、牌照或股东背景。
- 分析公司的经营表现，包括营收、利润、毛利率、净利率、ROE、现金流、资产负债情况等。
- 分析公司近期重要事件，如并购、重组、股权变化、重大合同、监管处罚、业绩预告等。
- 判断公司基本面是否改善、恶化或仍需观察。

4. 估值情况
- 结合公司盈利状态选择合适估值方法。
- 若公司盈利稳定，可分析 PE、PEG、PB、EV/EBITDA 等指标。
- 若公司亏损或周期波动大，应说明 PE 失效，并重点分析 PB、PS、市值、净资产、现金流、重整价值或资产价值。
- 与同行公司或历史估值区间进行对比。
- 判断当前估值是偏高、合理、偏低，还是主要由题材和预期支撑。

5. 投资风险分析
- 分析行业风险、政策风险、经营风险、财务风险、估值风险、市场情绪风险。
- 如适用，补充退市风险、ST风险、商誉减值风险、债务风险、诉讼风险、监管风险、客户集中风险、技术替代风险。
- 对每项风险说明可能造成的影响。

6. 风险控制
- 给出仓位控制建议。
- 给出跟踪指标，例如营收增速、利润率、现金流、订单、负债率、公告、监管进展等。
- 给出止损或退出条件，例如业绩继续恶化、重大利空、估值过高、核心逻辑被证伪等。
- 区分适合长期投资、波段参与、事件驱动交易或仅适合观察。

7. 综合判断
- 总结公司投资逻辑、主要看点和最大风险。
- 明确说明当前更适合积极配置、谨慎跟踪、等待拐点，还是规避。
- 用客观、克制、专业的语气，不要夸大收益，不要使用煽动性语言。

写作要求：
- 使用中文。
- 报告结构清晰，可使用小标题和分点。
- 尽量引用最新公开资料。
- 所有财务数据必须注明时间口径。
- 如果缺少最新数据，请明确说明“公开资料有限”或“需进一步核实”。
- 不要编造数据。
- 不要给出保证收益或确定性买卖建议。
- 结尾必须加上：“本文仅供研究交流，不构成任何投资建议。”

请分析的股票是：{股票名称或代码}
```
